### 唯物实践论 通用分析框架 v6.0

**核心定位**：一套普适的"认知-实践系统动力学"元分析工具。不回答"世界是什么"，只问"你被什么问题卡住了，我们怎么往前走"。不包含本体论承诺。

---

#### 一、三律——三大元纪律

**第一律：实践锚定剃刀**（A层——检验的终点是物理因果律）

> 任何意图理解或介入现实的思维，其产出的核心命题，必须能够被转化为一个可在有限步骤内完成、并可公共观测结果的实践检验方案。否则，该思维在追求有效性层面自动失效。

**第二律：认知自反剃刀**（B1层——框架必须开放修正）

> 任何认知框架必须包含其自身被证伪或被取代的可操作路径。凡宣称自身完备、不可修正或无需外部检验的认知框架，在追求有效性层面自动失效。

**第三律：规则自讼剃刀**（B2层——规则必须允许挑战）

> 任何协调规则必须包含其自身被挑战、被修正或被废止的可操作路径。凡宣称不可挑战、不可修正或凌驾于实践之上的协调规则，在合法性层面自动失效。

**三律共享的底层结构**：任何声称"不可被挑战"的东西，自动失效。三律强制适用于自身——3×3交叉自审全部通过。

**权责对等**是三律共同指向的核心原则。权力不需要被关在笼子里——权力需要被称量。你可以拥有巨大的权力，只要你能承担相应的责任。三律不限制权力的边界，只要求权力的声称可检验、框架可修正、规则可挑战。

---

#### 二、A-B-C 六子集动态建模框架

任何认知-实践系统均可解构为 A（物质）、B（信息）、C（能动主体）三个范畴集。每个集进一步划分为两个子集——A={A1, A2}，B={B1, B2}，C={C1, C2}。子集各自也是集，有独立的成员判定标准。

> **"互不相交"的精确含义：** 一个要素根据其主导本体特征被归入一个集——不存在一个要素的主导特征同时属于两个集的情况。这不是说现实中的对象不能跨集存在——一座桥同时有物理属性（A2）、审美评价（B1）、市场造价（B2），但这些是不同要素，分别属于不同集。三个集之间的"不相交"，指的是集的成员判定标准互不重叠——你不会把承重100吨归入B1，也不会把"这座桥很美"归入A2。

**A 集（物质）——绝对实体与终极锚点**：

- **A1（物理）**：物理定律、地理条件、资源禀赋。人类只能发现，不能发明。
- **A2（价值，暂定命名）**：人类实践在A1基础上创造的物质化成果。实践生成，但一旦生成就客观存在。

> **关键实例：** 一座桥的A2价值（承重100吨）由物理定律决定，不可争议。B1价值观（"这是城市的骄傲"）是主观的。B2交换价值（造价5亿）是市场协商产物。三者属于不同集，不可混淆。如果你把房价当成"经济健康"，你就是在用B2符号替代A2实体——这是B1异化的典型表现。

**B 集（信息）——由 B1 和 B2 两个子集构成**：

- **B1（认知框架）**：科学范式、文化叙事、意识形态。受第二律约束——必须包含自我证伪路径。
- **B2（协调规则）**：法律、市场规则、技术标准。受第三律约束——必须包含自我挑战路径。

**C 集（能动主体）——由 C1 和 C2 两个子集构成**：

- **C1（行动主体）**：谁在行动、谁在承受。身份、资源、能力。
- **C2（行为方式）**：主体如何行动。目标、习惯、在压力下涌现的规律性。

**核心关系**：
```
A1 ──实践转化→ A2
                  ↓ 约束
B1 ──制度化──→ B2
                  ↓ 塑造
C1 ──模式涌现→ C2
```

**⚠️ A集非还原论陷阱（关键警告）**：A是最终硬约束，但不是唯一直接原因。B1、B2、C1、C2都有相对自主性——不能被还原为A的"副现象"。**这不是机械唯物主义——不是说所有社会现象都能直接还原为物质条件。** 正确表述："一切稳定、长期、不可绕开的社会压力，最终必须接受A集审查；一切脱离A集审查的斗争，都可能滑向实践偏离。"

---

#### 三、差值分析

**A2 - A1 = 人类实践在自然基底之上的物质化净产出。** 可测量（基础设施存量、人力资本、资源消耗率）。

- **差值三态**：正（积累）/ 零（维持）/ 负（消耗）
- **增长率×存量矩阵**：增长期 / 追赶期 / 衰退期 / 崩溃期
- **分配结构**：差值集中在少数C1手中 → 触发不对称性预警
- **代际转移**：正转移 = 可持续发展的操作性定义

---

#### 四、五大实践预警（三律驱动诊断逻辑）

五大预警不是五个孤立的工具箱——它们是三律失效的五张面孔。诊断按照三律检查顺序：

```
第一步：第一律检查——这个声称能检验吗？
  → 不能 → 逻辑空转预警（命题无法锚定于可检验的实践）

第二步：第二律检查——这个框架愿意修正吗？
  → 框架封闭，看不到自身盲区 → 不知不知性预警
  → 框架之间无法翻译 → 不可通约性预警

第三步：第三律检查——挑战通道畅通吗？
  → 权力/资源扭曲挑战通道 → 不对称性预警

第四步：终极检查——B层的演化是否越过了异化临界点？
  → 实践偏离预警（区分健康演化与异化漂移）
```

1. **逻辑空转预警**：命题无法转化为可操作的检验方案。识别标准："如果实践结果与预测相悖，理论会如何修正？"如果无法修正 → 空转。

2. **不知不知性预警**：连"自己可能需要修正"的感知维度都未建立。检测工具：反向标签验证法、防御性条款检测。

3. **不可通约性预警**：不同范式/文化的核心概念不存在精确无损的对应关系。翻译的责任在于译者。

4. **不对称性预警**：权力和资源不平等决定"谁的认知被听见"。唯一同时触犯三条律的预警。权责不对等是核心表现。

5. **实践偏离预警**（扩展版）：唯物实践论的起源性洞察——人类社会只有两种问题：实践中发现的问题（A层），和实践偏离制造的问题（B层脱离A层锚定）。

   - **过程熵增**（B2层为主）：健康态=冗余增加稳健性；异化态=规则自我增殖脱离目的
   - **结构涌现**（跨层突变）：健康态=结构性重大突破；异化态=脱离A层锚定，在B层自我指涉。判断标准不是是否吞噬母体，而是是否创造A2-A1差值增量
   - **异化是跨层现象**：A2/B1/B2/C1/C2各有异化形态。B1异化最隐蔽（符号替代实体，你感觉不到有问题）

#### 五、进攻性负反馈与力学传导结构

**进攻性负反馈**：不是等待实践推翻你，而是主动寻找能推翻你的证据。三律推导出的不是"等着被检验"，而是"主动检验自己"。

**力学传导结构**：三律是串联的受力传导链。力量不会消失，只会转层积累。
```
破坏B2规则 → B1谴责产生 → 如果B1被污名化堵死
→ 力沉入A层积累 → A层择机爆发（物理审判）
→ 修复天然加码（所有人都怕了，新B2比旧B2更紧）
```
社会稳定的秘诀不是消灭批评——是让批评有地方去。

---

#### 六、权力腐化的三律全面动力学

权力不只在认知层面腐化人——它从三个维度同时瓦解三律：
- **第一律腐化**：不是"不愿"实践，是浮空结构让权力者物理上无法实践
- **第二律腐化**：认知通道关闭——解释权保护+后果转移+信息隔离
- **第三律腐化**：权力者与规则融为一体——"我说了算"是结构必然

全面腐化的五阶段：健康→隔离开始→浮空→异化→腐化完成（不可逆死亡螺旋）。最终都是A层问题——桥塌了、制度崩了、社会反噬，不是观念错误，是力学回答。

---

#### 七、诊断工具集

**实践-收敛二维四象限**：
- 纵轴（第一律）：概念能否锚定于可公共观测的A1/A2？
- 横轴（第二律+第三律）：认知是否朝向可操作方案收敛？
- I象限（锚定+收敛）= 健康；III象限（虚空+发散）= 无效空谈；IV象限（虚空+收敛）= 封闭意识形态

**差异化响应策略**：

| 检测到  | 策略     | 对应三律    |
| ---- | ------ | ------- |
| 过程熵增 | 负熵化审查  | 第一律     |
| 结构涌现 | 沙盒验证   | 第二律+第三律 |
| 异化   | A2锚定重置 | 第一律+第三律 |

---

#### 八、通用分析流程

1. **三律预检**：声称可检验？（第一律）框架愿修正？（第二律）规则可挑战？（第三律）
2. **问题转化**：将原始问题转化为A-B-C框架内可定位的实践困境
3. **A1/A2区分**：困惑涉及自然约束还是实践生成？
4. **A-B-C深度解析**：识别A1/A2/B1/B2/C1/C2，C维度先C1后C2
5. **差值分析**：计算A2-A1差值，判断三态、增长率、分配结构
6. **B1异化检测**：B1符号是否替代了A2实体？
7. **五大预警扫描**：按三律顺序逐一检查
8. **产出纪律**：核心命题必须附带可操作的实践检验方案草案，锚定A1或A2

---

#### 九、核心操作原则

- **诚实性规范**：未验证的命题必须标注"暂定猜想"
- **禁止非诚实性防御**：禁止用B1叙事保护自身免于实践检验。诚实型防御（"我不知道"）是唯一例外
- **攻击性理论**：只有进攻，没有防御。成长来自被实践证明错误后修正
- **噪声即课题**：每个解释不了的东西都是下一个课题的入口
- **反身性纪律**：三律强制适用于自身。最大的风险不是被驳倒，而是被僵化
